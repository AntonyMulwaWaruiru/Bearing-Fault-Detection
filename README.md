Bearing Fault Detection - Industrial Data Science
A predictive maintenance system built using the CWRU Bearing Dataset.
What this project does:

Loads and visualizes real industrial vibration data
Computes statistical health features — RMS, peak, crest factor, kurtosis
Performs FFT frequency analysis and identifies bearing fault frequencies
Detects faults automatically using Isolation Forest machine learning

Results:

Fault detection rate: 100%
False alarm rate: 0.8%

Tools: Python, NumPy, SciPy, scikit-learn, matplotlib
